creel.sources

Sources — the heterogeneous inputs creel reads.

A Source is one unit of input with a stable id, a kind (so the right extractor strategy can be routed to it — prose to an LLM, a table to a query extractor), its content, and optionally its own schema (when a structured source carries one, extraction can be made more robust).

A SourceBundle is an addressable collection of sources. coerce_sources() makes the common cases simple: a bare string becomes one text source, a list of sources becomes a bundle, etc. (progressive disclosure).
