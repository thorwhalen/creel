creel.ingest.protocol

The ingestion contract: a Loader turns a file into a Source.

Per decision D-OP7 (report R13), ingestion is upstream of extraction: files → Sources. A Loader is a callable (path, *, source_id) -> Source that emits LLM-friendly content (Markdown/text for prose, row dicts for tables, parsed objects for JSON) and records enough metadata that downstream extractors can ground each value back to a page/cell/char-span.

Loaders are callables behind a Protocol (composition over inheritance), mirroring the Extractor/Verifier strategy pattern. Route-by-format lives in creel.ingest.loaders; the public entry point is creel.ingest.ingest().
